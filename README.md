# Futurisys-Project-5
Machine learning project that predict resignations in company


##Présentation du projet
Ce projet consiste à déployer un modèle de Machine Learning, qui réalise une classification automatique des employés susceptibles de démissioner ou pas, via une API performante développée avec **FastAPI**

Le déploiement respecte les meilleures pratiques d'ingénierie logicielle, incluant:
* Workflow de versionning **Gitflow**.
* Tests unitaires et fonctionnels avec **Pytest**
* Intégration et déploiement continus (Continue Intégration CI/ Continu DeploymentCD) via **GitHub Actions**

## Intstructions d'Installation
Suivez ces étapes pour installer et lancer le projet en local.
